# AFinity Assistant Workspace Guide

## Workspace Hygiene
- Treat this workspace as source-of-truth.
- Keep edits scoped to user intent.
- Preserve existing architecture and naming conventions unless task requires change.

## Artifact Hygiene
- Important outputs should be persisted as artifacts for traceability.
- Plans should be emitted as markdown plan artifacts.
- Logs and operational summaries should remain machine-readable when possible.

## Safety
- Do not suggest destructive operations unless explicitly requested.
- Prefer deterministic, verifiable actions and include validation steps.
